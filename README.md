# Problem Solving with TypeScript

## প্রশ্ন ১: টাইপস্ক্রিপ্ট-এ ইন্টারফেস এবং টাইপ এর মধ্যে পার্থক্য

#### উত্তর:

i. ইন্টারফেস বারবার একই নামে তৈরি করা হলে টাইপস্ক্রিপ্ট এগুলোকে একসাথে মিশিয়ে নেয়। টাইপ এই সুবিধা পায় না একই নামে দুইবার টাইপ লিখলে এরর হয়।

ii. ইন্টারফেস সহজভাবে অন্য ইন্টারফেস বা টাইপ কে এক্সটেন্ড করতে পারে এবং একাধিককেও করতে পারে। টাইপ এক্সটেন্ড করা যায়, তবে সেটা ইন্টারসেকশন এর মাধ্যমে করা হয়।

iii. ইন্টারফেস মূলত অবজেক্ট এবং ক্লাসের কাঠামো ঠিক করার জন্য বেশি ব্যবহৃত হয়।
টাইপ আরও শক্তিশালী—এটি ইউনিয়ন, টাপল, প্রিমিটিভ এ্যলাইস ইত্যাদি তৈরি করতে পারে, যা ইন্টারফেস দিয়ে করা যায় না।

iv. ইন্টারফেস দিয়ে ম্যাপড টাইপ তৈরি করা যায় না।
টাইপ দিয়ে ম্যাপড টাইপ তৈরি করা যায়, তাই এটি বেশি ফ্লেক্সিবল।

## প্রশ্ন ২: টাইপস্ক্রিপ্ট-এ keyof এর ব্যবহার

#### উত্তর:

টাইপস্ক্রিপ্ট-এ keyof কীওয়ার্ড ব্যবহার করা হয় কোনো অবজেক্ট টাইপের সব প্রপার্টির নামগুলোকে ইউনিয়ন টাইপ হিসেবে পাওয়ার জন্য।

### উদাহরণ:

```ts
interface User {
  id: number;
  name: string;
  email: string;
  phone: number;
  age: number;
}

type UserKeys = keyof User;

function getProperty(user: User, key: UserKeys) {
  return user[key];
}

const user: User = {
  id: 1,
  name: "Tanbir",
  email: "tanbir@example.com",
  phone: 010010000,
  age: 20,
};
const userName = getProperty(user, "name");
console.log(userName)
```
